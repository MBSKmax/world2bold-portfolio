## üì± Image Optimization

### Before Uploading:

1. **Resize to specifications**
2. **Compress size**:
   - Use Tinypng.com (PNG/JPG compression)
   - Or Imagecompressor.com
3. **Format**:
   - SVG for illustrations (vector, smallest size)
   - PNG for transparent backgrounds
   - JPG for photographs (smaller file size)

### Quick Optimization Commands:

```bash
# Using ImageMagick (if installed):
convert project-1.png -resize 400x300 project-1-optimized.png
```
